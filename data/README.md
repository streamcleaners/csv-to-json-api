# UK Tariff Dummy Data

Dummy CSV data files modelled on the open datasets published by the
Department for Business and Trade (DBT) via
[data.api.trade.gov.uk](https://data.api.trade.gov.uk).

These files are **entirely fictional** and are intended for development,
testing, and demonstration purposes only. They are not a representation of
actual UK trade policy.

## Files

| File | Description |
|------|-------------|
| `commodities.csv` | Commodity code hierarchy with descriptions |
| `measures_on_declarable_commodities.csv` | Duty measures on 10-digit declarable codes |
| `trade_quotas.csv` | Tariff rate quotas with volumes and fill rates |
| `preferential_measures.csv` | Preferential duty rates under FTAs and trade arrangements |
| `commodity_code_changes.csv` | Historical changes to commodity codes |
| `geographical_areas.csv` | Geographical area codes used in tariff measures |
| `measure_types.csv` | Reference data for measure type codes |
| `certificates_and_licences.csv` | Document codes required for specific commodities |
